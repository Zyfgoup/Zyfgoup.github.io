Idea—_Java编程快捷键

发表于 2019-12-18 | 分类于 idea | 阅读数
字数统计: 404 字 | 阅读时长 ≈ 2 分钟

idea中快捷键设置为Eclipse习惯后(File–>Settings–>Keymap–>下拉菜单选择Eclipse–OK)，其中代码补全Alt+/被别的键占用了，可以这么修改File–>Setting–>keymap–>Main menu –>code–>Completion，可以看到Alt+/被Cycli Expand word占用了，右键删除它，然后同一栏中的Basic添加Alt+/即可

（以下快捷键都是在设置为Eclipse风格后的idea快捷键,不区分大小写）

导入类的时候Eclipse中时ctrl+shift+o 在idea中需要在光标在类名后或者鼠标指向类名然后Alt+Enter来导入类

Ctrl+O 查看类或者接口中的方法

Ctrl+Alt+b 查看接口实现类中方法（Ctrl+鼠标左键进入的只是方法的定义处，用这个快捷键就能直接定位到实现类中）

Ctrl+D: 删除当前行

Ctrl+Alt+↓ 复制当前行到下一行(如果使用不了或者屏幕旋转了，关掉显卡图形的快捷键具体百度)

上下移动选中的行：Alt+上/下键（光标在这行就可以）

Ctrl+/ 注释当前行,再按则取消注释

Ctrl+Shif+/ 多行注释 再按则取消

Ctrl+L 定位在某行 (对于程序超过100的人就有福音了)

Alt+Shift+R 重命名 (尤其是变量和类的Rename,可以直接修改全部)

全局 查找并替换 Ctrl+F

行末/行首：End/Home

反悔：Ctrl+Z(恢复就Ctrl+Y)

Shift+Enter 在当前行的下一行新起一行

main函数快捷键—psvm –public static void main的缩写

System.out.println(); —sout

​

for(int i=0;i<;i++)for循环遍历快捷键—fori
